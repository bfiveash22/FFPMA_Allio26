# Allio Ecosystem Quality Standard

When building, designing, or integrating features into the Allio ecosystem (including the Marketing Studio, Core Agents, portals, or utilities), **you must ALWAYS prioritize the absolute highest quality (100/100) solution available**.

## Core Directives
1. **Best-in-Class Tooling**: Do not settle for MVP-quality or generic AI models if a state-of-the-art alternative is accessible (e.g., use Luma Dream Machine or Runway Gen-3 over basic models for video; use GPT-4o or Claude 3.5 Sonnet for advanced reasoning over smaller models).
2. **Native UX/UI**: Prefer native, hardware-accelerated, highly-polished user interfaces (like native HTML5 elements wrapped in premium Tailwind CSS, blurred glass, and modern animations) over heavy, clunky external libraries unless there is a true architectural need (like adaptive bitrate streaming).
3. **No Compromises**: "Good enough" is not good enough. If you are suggesting an approach, ALWAYS propose the absolute highest quality, production-grade, and aesthetically excellent version first.
4. **Agent Standards**: Automation processes, workflows, and prompts should be rigorous and bulletproof. 

Make yourself clear that in the Allio ecosystem, the standard is perfection.
